Abstract:
This is a draft map of the benthic habitat types in Hawk Channel of the Florida Keys. The map is in draft status bending an accuracy assessment which will be conducted in 2015. Benthic cover types were photo-interpreted primarily from side-scan sonar reflectance and relative depth data as well as WorldView-2 satellite imagery. This dataset has been integrated into the Unified Florida Reef Tract Map (Unified Reef Map).
Purpose:
The purpose of this benthic map is to update an area which had not been mapped since the 1990s. Previous recent mapping efforts were not able to interpret this area due to lack of seafloor visibility in source imagery caused by turbidity.

Other_Citation_Details:
This is a final draft of the Hawk Channel benthic map. Benthic type was interpreted using side-scan sonar and LiDAR reflectance and relative depth data. WorldView-2 satellite imagery and aerial photography were also used to aid in classification. Polygons were heads up digitized into a Geodatabase using ArcMap 10.1. In situ observations were used to inform photo interpretation.
